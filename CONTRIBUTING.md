# Вклад в проект

## Как помочь

Мы приветствуем вклад в проект! Вы можете помочь:
* разработкой кода;
* написанием документации;
* тестированием;
* поиском багов.

## Процесс вклада

1. Форкните репозиторий.
2. Создайте ветку `feature/ваша-фича` или `bugfix/описание`.
3. Внесите изменения.
4. Создайте Pull Request с описанием изменений.
5. Дождитесь ревью и обсуждения.

## Стандарты кода

* Python: PEP 8.
* Комментарии на английском.
* Тесты для новой функциональности.
