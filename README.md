# СемантикМеш 2.0

Экосистема семантических технологий с поддержкой оффлайн‑режима.

## Описание

«СемантикМеш 2.0» — платформа для:
* семантического анализа текста;
* создания и синхронизации семантических меток (в т. ч. оффлайн);
* генерации контента на основе меток;
* опциональной этической проверки.

## Ключевые особенности

* **Оффлайн‑работа:** создавайте и редактируйте метки без интернета.
* **Синхронизация:** автоматическая синхронизация при подключении к сети.
* **Генерация:** создание текстов на основе семантических карт.
* **Гибкость:** модуль этики включается по необходимости.
* **Мультиплатформенность:** веб, десктоп, мобильные приложения.

## Технологии

* Python 3.11
* FastAPI (бэкенд)
* spaCy, BERT (NLP)
* PostgreSQL (база данных)
* Redis (кеширование)
