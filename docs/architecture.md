# Архитектура системы

## Компоненты

1. **Веб‑интерфейс** (React.js) — взаимодействие с пользователем.
2. **API‑шлюз** (FastAPI) — приём запросов, маршрутизация.
3. **Семантический движок** (Python + spaCy) — анализ текста.
4. **Модуль меток** (Python) — управление офлайн‑метками.
5. **Генератор контента** (Python + GPT‑подобные модели) — создание текстов.
6. **МЭО‑модуль** (Python + Scikit‑learn) — этическая оценка.
7. **Хранилище данных** (PostgreSQL + IPFS) — хранение текстов и результатов.
8. **Кеш** (Redis) — ускорение обработки повторяющихся запросов.
9. **Очередь задач** (Celery + RabbitMQ) — асинхронная обработка.
10. **Панель администратора** (Django) — управление настройками.

## Интеграции

* облачные платформы (AWS, GCP, Yandex Cloud);
* корпоративные системы (CRM, ERP);
* социальные сети (через API);
* образовательные платформы (Moodle, Canvas).
