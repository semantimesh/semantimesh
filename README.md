# SemantiMesh

Децентрализованная офлайн‑сеть с семантическим сжатием и блокчейном.

## О проекте

SemantiMesh решает проблему связи без интернета через mesh‑сети с гибридным сжатием данных.

## Ключевые особенности

- Офлайн‑чат для групп до 100 человек
- Передача данных IoT с 15‑кратным сжатием
- Стимулирование пользователей через токен $SEMANTI$
