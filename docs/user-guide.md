# Руководство пользователя

## Начало работы

1. Зарегистрируйтесь на платформе.
2. Загрузите текст для анализа.
3. Просмотрите автоматически созданные семантические метки.
4. Отредактируйте метки при необходимости (в т. ч. в оффлайн‑режиме).
5. Сгенерируйте контент на основе меток.
6. При необходимости включите этическую проверку.
7. Сохраните и экспортируйте результат.

## Режимы работы

* **Базовый:** только семантический анализ и метки.
* **Продвинутый:** с генерацией контента.
* **Экспертный:** полная функциональность + этика.

## Синхронизация меток

1. Создавайте метки без интернета — они сохранятся локально.
2. При подключении к сети начнётся автоматическая синхронизация.
3. Конфликты разрешаются автоматически (последняя временная метка побеждает).

## Экспорт данных

Поддерживаемые форматы:
* JSON;
* CSV;
* XML;
* PDF (аннотации).
