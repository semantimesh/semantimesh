SemantiMesh 2.0

Автономный mesh‑блокчейн с семантическим сжатием и офлайн‑платежами

Обзор проекта

Ключевые особенности

  • Работа без интернета через BLE, Wi‑Fi Direct, LoRa
  • Семантическое сжатие данных (экономия трафика в 10–15×)
  • Задержка подтверждения: ≤ 3 с в кластере
  • Энергопотребление: ≤ 8 % батареи/час
  • Размер блока: ≤ 1 КБ (до 50 транзакций)
