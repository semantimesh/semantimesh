# СемантикМеш 2.0: блокчейн на mesh‑сети

Децентрализованная платформа для семантического анализа с поддержкой офлайн‑режима.

## Ключевые особенности

* **Mesh‑сеть:** одноранговое взаимодействие без центрального сервера
* **Блокчейн:** распределённый реестр для хранения меток и истории изменений
* **Оффлайн‑работа:** создание и синхронизация меток без интернета
* **Токеномика:** стимулы для участников сети через токен $SEM$

## Архитектура

Система состоит из:
1. **Mesh‑модуля:** обнаружение узлов, передача данных
2. **Блокчейн‑модуля:** консенсус, хранение транзакций
3. **Модуля синхронизации:** разрешение конфликтов при восстановлении связи
