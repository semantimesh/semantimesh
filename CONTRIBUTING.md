# Вклад в проект

Мы приветствуем вклад в развитие «СемантикМеш 2.0».

## Как внести вклад

1. Форкните репозиторий.
2. Создайте ветку для вашей функции (`git checkout -b feature/AmazingFeature`).
3. Зафиксируйте ваши изменения (`git commit -m 'Add some AmazingFeature'`).
4. Отправьте ветку в удалённый репозиторий (`git push origin feature/AmazingFeature`).
5. Откройте Pull Request.

## Требования

* Следуйте стилю PEP 8.
* Добавляйте тесты для новой функциональности.
* Обновляйте документацию при необходимости.

## Вопросы?

Свяжитесь с командой через Issues.
