# SemantiMesh Whitepaper

**Версия 2.4** | Март 2026

## Abstract
SemantiMesh — децентрализованная модульная беспроводная сеть, предназначенная для надёжной оффлайн-передачи информации и платежей в условиях отсутствия традиционного интернета.

AI-агенты преобразуют контент в компактные семантические метки, передают их по беспроводным каналам, а на приёмной стороне генерируют полноценный контент. Система построена на принципах модульности, автономности и естественной ценности для участников.

## 1. Технология семантических меток

Метка — это семантическое ядро контента. Вместо передачи тяжёлых файлов система передаёт математическое описание смысла, структуры и ключевых характеристик.

**Пример реального сжатия:**
- Оригинальное видео 4K длительностью 30 секунд — 300–800 МБ
- Семантическая метка — 2–40 КБ (сжатие в тысячи раз)

**Состав метки:**
- Латентные токены — компактное представление кадров, движения, звука и освещения
- Семантический промпт — точное описание сцены, действий, тона и контекста
- Параметры стиля, эмоции и динамики
- Криптографический хэш для проверки целостности

**Процесс работы (полностью оффлайн):**
1. Локальный AI-энкодер на устройстве отправителя преобразует контент в метку.
2. Метка передаётся по беспроводной сети.
3. AI-декодер на приёмнике генерирует контент, максимально близкий к оригиналу по смыслу и восприятию.

Эта семантическая архитектура позволяет эффективно работать даже на низкоскоростных каналах.

## 2. SemantiPay — шлюз оффлайн-платежей

SemantiPay — это полностью децентрализованный протокол платежей токеном $SEMANTI, работающий без интернета.

**Как это работает:**
- Пользователь создаёт и подписывает транзакцию локально.
- Транзакция преобразуется в семантическую метку.
- Метка передаётся по беспроводной сети с использованием store-and-forward.
- На приёмной стороне платеж подтверждается локально (optimistic validation).
- При появлении любого узла с доступом к интернету происходит окончательное settlement на блокчейне.

SemantiPay создаёт настоящую экономику в отключённых условиях — платежи работают так же естественно, как и передача сообщений.

## 3. Беспроводные протоколы и маршрутизация

Архитектура модульная, поэтому поддерживает разные беспроводные протоколы как WASM-плагины:

- **LoRa** — базовый модуль для дальней связи (дальность до 50 км, низкое энергопотребление, store-and-forward).
- **Wi-Fi HaLow и BLE Mesh** — для средней дальности и плотных сетей.
- **Free-Space Optics (FSO)** — высокоскоростной модуль для мгновенной передачи меток.

**Умная маршрутизация меток:**
- Semantic-Aware Routing — решение о маршруте принимается с учётом смысла, приоритета и типа метки.
- Multi-Path Routing — критически важные метки отправляются по нескольким путям одновременно.
- Reputation-based Routing — предпочтение отдаётся надёжным узлам с высокой репутацией и стабильным энергоснабжением.

## 4. Ключевые компоненты системы

- **AI Contributor Swarm** — автономные агенты, которые оценивают вклад разработчиков и начисляют токены естественным образом.
- **Гибкий модуль этики** — пользователь самостоятельно выбирает уровень строгости (от полного отсутствия до строгих правил). Правила эволюционируют через DAO.
