# White Paper: СемантикМеш 2.0

## 1. Архитектура децентрализованной сети

«СемантикМеш 2.0» объединяет mesh‑сеть и блокчейн для создания устойчивой офлайн‑платформы.

**Ключевые компоненты:**
* **Mesh‑сеть:** одноранговая связь между устройствами без центрального сервера
* **Блокчейн‑реестр:** распределённое хранилище семантических меток и истории изменений
* **Модуль синхронизации:** механизм разрешения конфликтов при восстановлении связи

**Преимущества гибридной архитектуры:**
* работа без интернета
* устойчивость к цензуре и атакам
* криптографическая защита данных
* автоматическое восстановление целостности

## 2. Передача и синхронизация меток

**Процесс работы с метками:**
1. **Создание:** пользователь добавляет метку на локальном устройстве
2. **Локальное хранение:** метка сохраняется в локальной базе данных с цифровой подписью
3. **Офлайн‑распространение:** через mesh‑сеть метка передаётся соседним узлам
4. **Глобальная синхронизация:** при появлении интернета изменения записываются в блокчейн
5. **Консенсус:** узлы проверяют валидность меток и разрешают конфликты

**Гарантии:**
* неизменность истории изменений
* защита от подделки меток
* автоматическая репликация актуальных данных

## 3. Экономическая модель

Для стимулирования участия в сети вводится токен **$SEM$**:
* **За ретрансляцию меток:** узлы получают вознаграждение за передачу данных
* **За хранение блокчейна:** участники, сохраняющие полную копию реестра, получают бонусы
* **За валидацию:** активные узлы консенсуса получают долю от комиссий

**Применение токена:**
* доступ к премиум‑функциям
* голосование за обновления протокола
* оплата услуг внутри экосистемы

## 4. Оффлайн‑режим

В условиях отсутствия интернета:
* устройства формируют локальную mesh‑сеть
* изменения сохраняются локально с криптографической подписью
* при восстановлении связи запускается процесс консенсуса:
  * узлы обмениваются транзакциями
  * конфликтующие версии разрешаются автоматически
  * актуальная версия записывается в блокчейн
* гарантируется целостность данных и предотвращаются конфликты
